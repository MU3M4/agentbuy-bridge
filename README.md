# AgentBuy Bridge

Pay $0.005 USDC to instantly unlock direct Chinese supplier details for Western buyers.

Built for the Nano Payments on Arc Hackathon - Autonomous Commerce Flows track.

Demo Features:
- Gas-free nanopayments using Circle x402
- Unlocks 1688 factory links, ¥ prices, shipping times
- Designed to work with agents like ASDBUY
- Real tiny payments on Base Sepolia / Arc testnet

How it works:
Users pay $0.005 USDC → get instant supplier data (no high fees, no gas).
